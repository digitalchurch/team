# General Settings

Set the following general settings in the App Customizer.

- Side Menu: `{church}-more-links`
- Tab Menu: `{church}-tabs`
- **TRUE** Show tab menu on every page
- Auxiliary Menu: `-none-`
- **TRUE** Enable Media Playback
- **FALSE** Enable Gelocation
- **FALSE** Enable Camera
- Translation Files: `none`
- Offline Files: `none`
- Default Language: `English`
- Custom Javascript: `none`
- Header Logo: Set from Brand Folder
- **FALSE** Status Bar: Light text
- **FALSE** Right Side Menu
- **FALSE** RTL
- **TRUE** Hide share buttons
