### 1. Activate the Gravity Forms Cloudflare Turnstyle plugin

In the dashboard, visit the plugins page and make sure that Gravity Forms Cloudflare Turnstyle is active.

### 2. Setup Domain in Cloudflare

In cloudflare, navigate to the Turnstyle section in the account and add the domain to retreive the keys needed for the plugin.

### 3. Setup Cloudflare Turnstyle in Form Settings

You'll need to enter the keys into the Cloudflare Turnstyle settings for Gravity Forms to be able to use Turnstyle fields.

### 4. Add a turnstyle field to the form(s).

In each form, add a turnstyle field as the last field in the form.
