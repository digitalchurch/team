---
title: Troubleshooting perfmatters
---

You can quickly troubleshoot page load issues to see if they are related to perfmatters by using a URL Query Parameter in the URL. This can also be applied to speed testing tools like Page Speed Insights.

```html title="Load Page with Perfmatters Off"
https://yourdomain.church/?perfmattersoff
```

```html title="Load Page with Perfmatters JS Management Off"
https://yourdomain.church/?perfmattersjsoff
```

```html title="Load Page with Perfmatters CSS Management Off"
https://yourdomain.church/?perfmatterscssoff
```

```html title="Quickly Access the Script Manager"
https://yourdomain.church/?perfmatters
```
