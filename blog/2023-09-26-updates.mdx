# Software Updates

## Server Updates

Today, GridPane solved an issue we were having with our Failover Server for Digital Church. SSL Certificates were not working properly on the failover. Here's the response we got from them:

> Hi Mark,
We have fixed the issue with SSL.
â€‹The failsync server had issues with a broken symlink **/etc/resolv.conf** which needed to be fixed. 
We had to do a manual sync  between source and failsync site to make sure that the sites are synched

Source: https://support.gridpane.com/helpdesk/tickets/92057

Also, we setup a CORS policy for digitalchurch.app on this server. Here's the policy we set:

```bash title="~/www/digitalchurch.app/nginx/cors-main-context.conf"
more_clear_headers "Access-Control-Allow-Origin";
more_set_headers "Access-Control-Allow-Origin: *";
```
